# 💻 Algorithm Study

## 📄 How To Study
- 매일(주말 제외) 하나의 주제에 다같이 선정한 문제에 대해서 문제 풀이 진행
- 매일 오후 10시 반 스터디 진행 전까지 서로의 코드를 보며 리뷰 작성
- 모여서 리뷰에 대한 확인 및 merge
- 다같이 문제 선정
- [Github 활용법](./docs)
